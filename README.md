## What is this?

Poor Man's Loom is a simple screen recording app built to avoid paying for expensive subscriptions. Record your screen easily without worrying about monthly fees or data privacy concerns.

## Features

- Record your screen with a few clicks (and zero shipping time from overseas)
- free, no surprises in your credit card statement a month later

## Getting Started

```bash
# Clone this repo
git clone https://github.com/your-username/screeclip.git

# Install dependencies (and hopes and dreams)
bun install

# Run the app
bun run dev
```
